proposal for the project 2: 

ALGORITHMIC TRADING SIMULATOR WITH MONTE CARLO SIMULATION. 

description of what I am building: 
- a web based trading simulator using algorithms that allows users to learn, understand, and use simple trading strategies to simulate portfolio outcomes, embedded with monte carlo methods. 

Who is this for? 
- this tool is created for beginner traders that need guidance on strategies to perform under uncertainty. Instead of just relying on one outcome, the user can learn a range of possibilities which helps them understand risk and volatility, and the impact of the different trading strategies. 

CORE FEATURES: 
- strategy input: users can input trading rules which includes the return, volatility, number of trades and starting capital. 
- Monte Carlo simulation: The app could run hundreds or thousands of sumulated price paths to model different market outcomes. 
- Visualization of portfolio outcome: displays results such as average return, best/worst case scenario, and distribution of outcomes. 
- Interactive controls: sliders and input fields to adjust assumptionsin real time. 
- Reset functionality: users can clear inputs and rerun simulations without refreshing the page. 

Things that I do not know yet: 
- how to efficiently generate and run multiple monte carlo simulations in Javascript. 
- How to visualize simulation results clearly. 
- how to structure and store simulation data. 
- How to Balance performance with accuracy when running many simulations in the browser. 
- How to implement randomness correctly to simulate realistic market behavior.
